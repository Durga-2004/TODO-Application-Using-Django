# ToDo Application Using Django

This is a simple ToDo application built using Django.  
You can add daily activities, view them, and delete them after completion.

## Features
- Add tasks
- View all tasks
- Delete tasks
- Data is stored permanently in the database

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite Database

## Description
This project helps in managing daily activities.  
Users can add tasks and delete them once completed.
